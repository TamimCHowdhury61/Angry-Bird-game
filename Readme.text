                  Angry Birds Game

Welcome to the game of Angry Birds! LibGDX was used in the development of this project, which gives players an entertaining and engaging gaming experience. You will find instructions on how to launch the game and comprehend its components in this README.

Aspects
Different degrees of interactive gaming.
Options to change the game's controls and music, for example.
There are several screens for navigation, including settings and the main menu.

Requirements
You must have the Java Development Kit (JDK) installed on your computer in order to play this game.
For dependency management, use Gradle or Apache Maven.
The LibGDX library.
Getting Started




Make a clone of the repository: link: 
https://github.com/Tamim544/Angry-Bird-game.git
Launch the project: Go to the LibGDX website and follow the setup instructions.
Include assets: Make sure all of the necessary asset files—such as pictures and sounds—are in the right folders.
How to Run the Game
Launch the sdk apps in your desktop
Open the downloaded file in the apps 
Run the Main game class. The game window will open as a result.
To play the game and explore menus, use the  mouse.
Game Screens
The game consists of several screens:
Main Menu Screen: Start a new game or navigate to settings.
Game Screen: Play the game and complete levels.
Level End Screen: View results after completing a level.
Load Page: Class for the loading page
Settings Screen: Adjust game settings like musics and sound effects
Pause Screen: Pause the game and return to the main menu.
Bird Screen: yet to implement
Pig screen: yet to implement
Roadmap: View levels and progress.
Adding Screen: It is basically the Map screen 
Setting Screen 2: Access advanced settings.
